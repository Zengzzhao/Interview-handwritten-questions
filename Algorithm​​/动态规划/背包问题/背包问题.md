# 01背包

现有N件物品、最大容量为W的背包。第i件物品的重量是weight[i]，其价值是value[i] 。**每件物品只能用一次**，求解将哪些物品装入背包里价值总和最大。

一维dp下遍历时，外层for循环为物品（从前往后），内层循环为背包容量（从大到小，确保一个物品遍历所有背包容量时取到的是上一个物品一轮遍历完的状态，确保每个物品只用一次）

# 完全背包

现有N件物品、最大容量为W的背包。第i件物品的重量是weight[i]，其价值是value[i] 。**每件物品无限个（即可以放入背包多次）**，求解将哪些物品装入背包里价值总和最大。

一维dp下遍历时，外层for循环为物品（从前往后），内层循环为背包容量（从小到大，确保一个物品遍历所有背包容量时取到的是更新过后的值，确保每个物品可以重复使用）
